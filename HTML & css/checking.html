Introduction

Create an Egg Rate automatic price update website using this PHP script.

Features

  • Official NECC Egg Rate data
  • Automatic price update
  • 899+ Locations egg rate
  • No API required
  • No database required
  • Easy to setup
  • Advanced features
  • Dynamic content
  • 100% Responsive
  • Built with Bootstrap

Requirements

  • PHP 7.4
  • cURL must be enabled
  • allow_url_fopen must be enabled
  • Apache server

Installation

This is how you install our product.

  • Upload and extract zip file on your server
  • Open site_settings folder's site_config.php file
  • Add your License Key on $license_key
  • Change website general information
  • Change website theme and color
  • Open your website home page and refresh 2 time
  • Set every 30 minutes cron jobs https://example.com/?cron=update
  • After installation complete delete this documentation.html file
